# Aplicativo WePraise de Oração Diária

Desenvolvimento de um aplicativo para auxiliar na oração diária, com funcionalidades de: criação de pedidos de oração, acompanhamento de orações, definição de metas de oração, lembretes de oração e compartilhamento de orações.  

# Tecnologias utilizadas

- React
- TypeScript
- Vite
- Firebase

# Como rodar o aplicativo

- pnpm install
- pnpm dev
